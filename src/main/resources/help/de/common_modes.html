Operating modes

       

Filius has three operating modes.

Design mode

In this mode, the various elements of the network can be assembled on the workspace. This mode is the one selected by default when starting Filius.

To select this mode from one of the other two, use the F10 keyboard shortcut or click on the following button on the top bar:

Documentation mode

This mode allows you to add text boxes as well as rectangular blocks on the workspace. These elements do not influence the simulation but make it easier to read the network.

To select this mode from one of the other two, use the F9 keyboard shortcut or click on the following button on the top bar:

Simulation mode

This mode is used to simulate network operation. The simulation includes the exchanges of data between the different elements that make up the network, as well as the interactions with the applications of the virtual stations of the network.

To select this mode from one of the other two, use the F11 keyboard shortcut or click on the following button on the top bar:
