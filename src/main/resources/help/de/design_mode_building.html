Allgemeines

Im Entwurfsmodus können Sie ein Computernetzwerk aufbauen, indem Sie die verschiedenen Komponenten auf der Arbeitsfläche zusammenstellen. Außerdem können Sie einige Parameter dieser Komponenten konfigurieren.

Netzwerkaufbau

Wählen Sie mit einer Drag-and-Drop-Maus Komponenten am linken Rand aus und ordnen Sie sie auf der Arbeitsfläche an.

Halten Sie beim Ziehen und Ablegen einer Komponente die Alt- (oder Wahl-) Taste gedrückt, um sie am Raster einzurasten.

Um eine Station der Arbeitsfläche zu duplizieren, klicken Sie darauf, um sie auszuwählen, und verwenden Sie dann die Tastenkombination Strg + D.

Verbinden Sie die Komponenten mit Kabeln. Wählen Sie dazu das Kabelwerkzeug aus, indem Sie auf das entsprechende Symbol am linken Rand klicken,

oder die Tastenkombination K verwenden, und klicken Sie dann nacheinander auf zwei zu verbindende Komponenten.

Um das Kabelwerkzeug abzubrechen, verwenden Sie Ihre Wahl: die Tastenkombination K, die Esc-Taste oder einen Rechtsklick mit der Maus.

Um eine Komponente oder ein Kabel zu löschen, klicken Sie darauf und drücken Sie dann die Entf-Taste der Tastatur.

Um alle mit einer Komponente verbundenen Kabel zu löschen, verwenden Sie das Kontextmenü (Rechtsklick).

Netzwerkkomponenten konfigurieren

Jede Komponente verfügt über eine Reihe von Parametern, die über den Eingabebereich im unteren Teil des Fensters definiert werden können.

Um den Eingabebereich aufzurufen, doppelklicken Sie auf eine Komponente oder ein Kabel.

Wenn der Eingabebereich angezeigt wird, zeigt ein einfacher Klick auf eine andere Komponente deren Parameter an.

Der Eingabebereich wird geschlossen, wenn Sie auf einen leeren Bereich der Arbeitsfläche klicken.

Wenn mehrere Stations oder Switches ausgewählt sind, reduziert sich die Anzahl der änderbaren Parameter.

Weitere Informationen zu den Komponenten finden Sie im Menü oben in diesem Fenster.
