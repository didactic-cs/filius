Mode conception

Service DHCP

Le service DHCP attribue à tout poste qui en fait la demande une adresse IP et un masque de sous-réseau. Il peut éventuellement également fournir l'adresse IP d'une passerelle et l'adresse IP d'un serveur DNS.

Configuration

Adresse IP de départ / Adresse IP de fin

Ces deux adresses IP délimitent la plage des adresses IP qui seront attribuées aux postes qui en solliciteront. Ces adresses doivent faire partie du même sous-réseau que le serveur.

Pour les composants du réseau dont les adresses IP ne sont pas attribuées automatiquement par un serveur DHCP, il conviendra de s'assurer que les adresses utilisées ne sont pas dans la plage des adresses distribuées par le serveur.

Masque de sous-réseau

Ce champ en lecture seule rappelle le masque de sous-réseau du serveur. C'est également ce masque qui est communiqué aux clients en même temps que l'adresse IP demandée.

Réservations

Ce bouton permet d'accéder à une liste d'adresses IP réservées. Il s'agit de couples adresse MAC/adresse IP permettant d'attribuer une adresse IP spécifique à certains composants qui ont besoin de toujours disposer de la même adresse IP.

Dans le tableau, utiliser le menu contextuel (clic droit) sur les lignes pour ajouter, supprimer ou modifier un couple d'adresses réservées.

Les adresses IP réservées ne doivent pas faire partie de la plage des adresses IP distribuées de façon automatique.

Passerelle / Serveur DNS

Ces deux champs permettent de spécifier l'adresse IP d'une passerelle ainsi que l'adresse IP d'un serveur DNS qui seront communiqués aux postes clients.

Identiques aux paramètres du serveur

Lorsque cette case est cochée, le serveur utilise ses propres paramètres de passerelle et de serveur DNS plutôt que ceux définis ci-dessus.

Activer le service DHCP

En cochant cette case, le service DHCP est rendu actif sur ce poste.

À chaque fois que le mode simulation sera activé, les postes configurés comme clients DHCP solliciteront un serveur DHCP pour obtenir une adresse IP, un masque de sous-réseau, l'adresse IP d'une passerelle et l'adresse IP d'un serveur DNS.

Un poste dont le service DHCP est activé est aisément identifiable grâce à son icône.
