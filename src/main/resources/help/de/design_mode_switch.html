Mode conception

Switch

Le composant Switch, ou commutateur, permet d’interconnecter plusieurs composants entre eux au sein d’un même réseau local. Sa fonction est de faire circuler les datagrammes en utilisant les adresses MAC des composants du réseau local.

Chaque commutateur de FILIUS possède 24 ports qui permettent de raccorder autant de composants, y compris d'autres commutateurs.

Les commutateurs de FILIUS sont non manageables. Ils n'ont donc pas d'adresse IP.

Configuration

Nom

Le nom peut être utilisé pour différencier plusieurs commutateurs, mais des commutateurs distincts peuvent également porter le même nom. Ce nom n'a aucun effet sur la simulation.

Afficher comme un nuage

Cette option permet de changer l'apparence du commutateur sur l'espace de travail. Elle n'a aucun effet sur la simulation.

Interfaces

Cette liste affiche la correspondance entre les ports du commutateur et les composants raccordés.

Sélectionner une ligne du tableau aura pour effet d'afficher le câble correspondant en bleu.
