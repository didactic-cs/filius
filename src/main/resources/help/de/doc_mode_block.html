Allgemeines

In diesem Modus können Sie visuelle Elemente hinzufügen, die das Lesen des Netzwerkdiagramms erleichtern. Diese Elemente haben keinen Einfluss auf die Simulation.

Im Dokumentationsmodus werden Gliederungsfläche und Textfelder vor den Netzwerkkomponenten angezeigt, um deren Positionierung zu erleichtern. In anderen Modi werden sie hinter den Komponenten angezeigt.

Gliederungsfläche

Wählen Sie mit einer Drag-and-Drop-Aktion (Maus) die Gliederungsfläche aus am linken Rand und legen Sie sie auf der Arbeitsfläche ab.

Dann verschieben oder skalieren Sie diese Fläche mit der Maus.

Um eine Gliederungsfläche zu duplizieren, klicken Sie darauf, um sie auszuwählen, dann verwenden Sie die Tastenkombination Strg + D.

Um eine Gliederungsfläche zu löschen, klicken Sie darauf und drücken Sie dann die Entf-Taste auf der Tastatur.

Verwenden Sie das Kontextmenü der Gliederungsfläche (Rechtsklick), um seine Farbe zu ändern.

Die Stapelreihenfolge der Gliederungsflächen kann über das Kontextmenü geändert werden. Diese kann nützlich sein, wenn sich zwei Gliederungsflächen überlappen.

Textfelder

Wählen Sie das Textfeld per Drag & Drop mit der Maus aus am linken Rand und legen Sie es auf der Arbeitsfläche ab.

Wenn ein Textfeld ausgewählt ist, wird es von einem blauen Rahmen umgeben. Ansonsten grenzt ein einfacher grauer Umriss den Bereich ab.

Um das Textfeld zu ändern, klicken Sie darauf. Der Text kann dann direkt bearbeitet werden.

Verwenden Sie den blauen Rahmen, um das Textfeld zu verschieben oder die Größe zu ändern.

Um ein Textfeld zu duplizieren, klicken Sie darauf, um es auszuwählen, dann verwenden Sie die Tastenkombination Strg + D.

Um das Textfeld zu entfernen, klicken Sie darauf, um es auszuwählen, dann verwenden Sie die Tastenkombination Strg + Entf.

Verwenden Sie das Kontextmenü, um die Schriftfarbe, -größe und -stil zu ändern.

Bei Bedarf kann die Stapelreihenfolge der Textfelder über das Kontextmenü geändert werden. Beachten Sie, dass Textfelder immer vor Gliederungsflächen angezeigt werden.
