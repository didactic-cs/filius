Mode simulation

Général

Ce mode permet de simuler le fonctionnement du réseau virtuel de FILIUS.

Les postes possèdent chacun un bureau sur lequel peuvent être exécutés des applications ou des services.

Les différentes fenêtres

En mode simulation, diverses fenêtres peuvent être utilisées en plus de la fenêtre principale.

  • La fenêtre d'aide ;
  • Les deux fenêtres de suivi des données qui permettent de visualiser les flux entrants et sortants des postes et des routeurs ;
  • Les fenêtres des tables SAT des commutateurs ;
  • Les fenêtres des bureaux des postes dans lesquelles sont affichées les fenêtres des applications virtuelles.

Un double-clic sur une zone vide de l'espace de travail de la fenêtre principale permet de repasser toutes les fenêtres devant la fenêtre principale.

Réduire la fenêtre principale réduit automatiquement toutes les autres fenêtres.
