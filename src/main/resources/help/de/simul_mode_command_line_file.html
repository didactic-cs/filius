Befehlszeile

Mit dieser Anwendung können Sie Befehle eingeben, um Dateien zu bearbeiten und netzwerkbezogene Operationen durchzuführen. Jeder Befehl muss gefolgt von seinen möglichen Parametern eingegeben und mit der Eingabetaste beendet werden.

Einige Befehle haben zwei Syntaxen.

Der Verlauf der eingegebenen Befehle ist mit den Auf- und Ab-Pfeilen auf der Tastatur zugänglich.

Allgemeine Befehle

help
?

Dieser Befehl ohne Parameter zeigt die Liste der verfügbaren Befehle an.

cls
reset

Dieser Befehl setzt die Anzeige des Befehlsfensters zurück.

exit

Dieser Befehl schließt das Anwendungsfenster.

Dateibefehle

Die Pfade, die von einigen Befehle verwendet werden, müssen das Trennzeichen " / " verwenden.

Das Zeichen " / " ist optional am Ende von Ordnerpfaden.

Absolute Pfade beginnen mit " / ".

Pfade, die nicht mit " / " beginnen, sind relativ zum aktuellen Arbeitsverzeichnis.

touch  datei-pfad

Dieser Befehl erstellt eine leere Datei.

Der Parameter datei-pfad muss der Name der zu erstellenden Datei sein. Dem Namen kann ein absoluter Pfad oder relativ zum aktuellen Ordner vorangestellt werden.

copy  quell-pfad  ziel-pfad
cp  quell-pfad  ziel-pfad

Dieser Befehl erstellt eine Kopie einer Datei.

Der Parameter quell-pfad muss der Name der zu kopierenden Datei sein. Dem Namen kann ein absoluter Pfad oder relativ zum aktuellen Ordner vorangestellt werden.

Der Parameter ziel-pfad muss der Name der zu erstellenden Kopie sein. Dem Namen kann ein absoluter Pfad oder relativ zum aktuellen Ordner vorangestellt werden.

move  quell-pfad  ziel-pfad
mv  quell-pfad  ziel-pfad

Dieser Befehl verschiebt oder benennt eine Datei um.

cat  datei-pfad
type  datei-pfad

Dieser Befehl zeigt den Inhalt einer Datei an.

mkdir  ordner-pfad

Dieser Befehl erstellt einen Ordner.

del  pfad
rm  pfad

Dieser Befehl löscht eine leere Datei oder einen leeren Ordner.

rmdir  ordner-pfad

Dieser Befehl löscht einen Ordner und seinen gesamten Inhalt.

cd  ordner-pfad

Mit diesem Befehl können Sie das aktuellen Arbeitsverzeichnis ändern.

pwd

Dieser Befehl zeigt den absoluten Pfad des aktuellen Arbeitsverzeichnisses an.

dir  ordner-pfad
ls  ordner-pfad

Dieser Befehl listet den Inhalt des Ordners auf, dessen Pfad angegeben ist. Wenn kein Parameter vorhanden ist, wird der Inhalt des Arbeitsverzeichnisses aufgelistet.

Netzwerkbefehle

ipconfig

Dieser Befehl zeigt die Konfiguration der Netzwerkschnittstelle an.

netstat

Dieser Befehl zeigt Informationen zu Netzwerkverbindungen an.

arp

Der ARP-Cache der Station ist eine Tabelle mit ARP-Einträgen. Jeder dieser Einträge ist ein Paar aus einer IP-Adresse und der entsprechenden MAC-Adresse. Der ARP-Cache wird verwendet, um die Auflösung von IP-Adressen in MAC-Adressen zu beschleunigen.

Ohne Parameter oder nur mit dem Parameter -a zeigt der Befehl arp den Inhalt des ARP-Cache an.

Zusammen mit dem Parameter -a gefolgt von einer IP-Adresse zeigt der Befehl arp nur den Eintrag an, der dieser Adresse zugeordnet ist.

Zusammen mit dem Parameter -d gefolgt von einer IP-Adresse entfernt der Befehl arp den Eintrag, der dieser Adresse zugeordnet ist, aus dem Cache.

Zusammen mit dem Parameter -d gefolgt vom Zeichen * löscht der Befehl arp alle Cache-Einträge.

Der Cache wird auch bei jedem Start der Simulation neu initialisiert.

arpsend  fiktive-ip  ziel-ip

Dieser Befehl sendet an eine Zielstation eine Korrespondenz zwischen der MAC-Adresse der Station, auf der der Befehl eingegeben wird, und einer beliebigen IP-Adresse. Dieser Korrespondenz wird der ARP-Tabelle der Zielstation hinzugefügt.

Dieser Befehl kann zu Testzwecken verwendet werden.

host  hostname

Dieser Befehl löst einen Hostnamen in eine IP-Adresse auf.

Der Parameter hostname entspricht dem Namen der Station, deren IP-Adresse gesucht wird.

route

Dieser Befehl zeigt die Routing-Tabelle der Station an.

ping  ip-adresse | hostname

Dieser Befehl testet die Netzwerkverbindung mit einer anderen Station.

Der Parameter IP-Adresse | hostname ist entweder die IP-Adresse oder der Hostname der zu testenden Station.

traceroute  ip-adresse | hostname

Dieser Befehl listet die Sprünge auf, die zum Erreichen einer Station erforderlich sind.

Der Parameter IP-Adresse | hostname ist entweder die IP-Adresse oder der Hostname der zu erreichenden Station.

tcpdump

Dieser Befehl wird verwendet, um den Netzwerkverkehr zu überwachen, indem jeder der Ethernet-Frames angezeigt wird, die in die Station eintreten oder diese verlassen.

Er zeigt dieselben Informationen wie das Datenaustauschfenster an.

Um diesen Befehl zu unterbrechen, geben Sie Strg+C ein.
