Mode simulation

Bureau

Chaque poste dispose d'un bureau virtuel sur lequel peuvent être exécutées des applications. Une seule instance de chaque application peut être exécutée par poste.

Pour afficher le bureau d'un poste, double-cliquer sur son icône ou utiliser son menu contextuel.

Installer et utiliser des logiciels

Les applications peuvent être installées à partir du bureau des postes. Pour cela, double-cliquer sur l’icône suivante :

Les icônes des applications installées apparaissent alors sur le bureau ainsi que dans le menu démarrer. Un double-clic sur une icône du bureau ou un simple clic dans le menu démarrer permet de lancer l’application correspondante.

Chaque application s’ouvre dans une fenêtre du bureau. Fermer la fenêtre d’une application, ou fermer la fenêtre du bureau ne ferme pas les applications. Seule la désinstallation permet de fermer une application.

Lorsque le mode change (pour le mode conception ou documentation), toutes les applications sont gelées. Elle redeviennent actives lorsque le mode simulation est réactivé.

État du réseau

En plaçant la souris au dessus de l’icône réseau située en bas à droite du bureau, l’adresse IP du poste apparait dans une infobulle.

Pour obtenir le paramétrage IP détaillé du poste, cliquer sur cette icône.

La modification des paramètres réseau du poste n'est pas possible en mode simulation. Il faut revenir au mode conception pour cela.
