Explorateur

Espace de stockage

Chaque poste dispose d'un espace de stockage virtuel dans lequel des dossiers peuvent être créés et des fichiers enregistrés. L'application Explorateur permet d'effectuer les opérations usuelles de renommage, déplacement, copie et suppression de ces fichiers et dossiers.

Il est également possible d'importer des fichiers réels de petite taille (150 Ko au maximum) dans l'espace de stockage ou d'exporter des fichiers virtuels en utilisant leur menu contextuel (clic droit).

L'espace de stockage de chaque poste est sauvegardé avec le projet.

Ouverture de fichiers

Un double-clic sur un fichier dont le format est reconnu permet son ouverture dans l'application associée, à condition que cette dernière ait été préalablement installée sur le poste.

Ainsi, les fichiers textes, css et xml sont ouverts dans l'éditeur de textes ; les fichiers html sont ouverts dans le navigateur web ; les images sont ouvertes dans l'afficheur d'images.
