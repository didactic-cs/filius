Mode simulation

Filius-Share

Cette application permet de partager des fichiers en pair-à-pair. L'application est à la fois client et serveur. Elle échange des fichiers avec les applications Filius-Share des autres postes. Deux postes qui échangent des fichiers entre eux sont qualifiés de voisins.

Réseau

Voisinage

Ajouter un voisin en spécifiant son adresse IP.

Utiliser le menu contextuel de la liste pour supprimer un ou tous les voisins.

Service

Le nombre maximal de voisins est le nombre maximal de postes auxquels le service pair-à-pair peut se connecter simultanément.

Cocher la case Activer le service pair-à-pair pour rendre ce dernier opérationnel.

Un poste dont le service pair-à-pair est activé est aisément identifiable grâce à son icône.

Rechercher

La recherche de fichiers s'effectue sur les noms, extensions comprises.

Par exemple, .png permettra d'obtenir toutes les images au format PNG.

La saisie de bon permettra d'obtenir tous les fichiers dont le nom contient bon.

Double-cliquer sur une entrée de la liste pour télécharger le fichier correspondant.

Utiliser le menu contextuel de la liste pour supprimer un ou tous les résultats de recherche.

Partager

Cet onglet affiche le contenu du dossier /peer2peer. Ce dossier contient les fichiers partagés. Les fichiers téléchargés sont également placés dans ce dossier et se trouvent donc de ce fait également partagés.

Double-cliquer sur une entrée de la liste pour ouvrir le fichier correspondant.

Le bouton Ouvrir le dossier /peer2peer permet d'ouvrir le dossier des partages dans l'explorateur de fichiers.
