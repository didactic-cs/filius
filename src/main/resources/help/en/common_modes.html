Général

Le mode conception permet d’assembler les différents composants d’un réseau informatique. Il permet également de configurer certains paramètres de ces composants.

Construction du réseau

À l’aide d’un glisser-lâcher à la souris, sélectionner des composants dans la marge de gauche et les disposer sur l’espace de travail.

Maintenir la touche Alt (ou Option) pendant le glisser-lâcher permet d'aligner le composant sur la grille.

Pour dupliquer un poste de l'espace de travail, cliquer dessus pour le sélectionner, puis utiliser le raccourci clavier Ctrl + D.

Raccorder les composants à l’aide de câbles. Pour cela, sélectionner l’outil câble en cliquant sur l’icône correspondante dans la marge de gauche,

ou bien à l’aide du raccourci clavier C, puis cliquer successivement sur deux composants à raccorder.

Pour annuler l'outil câble, utiliser au choix : le raccourci clavier C, la touche Echap du clavier, ou un clic droit à la souris.

Pour supprimer un composant ou un câble, cliquer dessus, puis appuyer sur la touche Suppr du clavier.

Pour supprimer tous les câbles raccordés à un composant, utiliser son menu contextuel (clic droit).

Paramétrage des composants du réseau

Chaque composant possède un ensemble de paramètres qui peuvent être définis à l'aide du panneau de configuration situé dans la partie inférieure de la fenêtre.

Pour faire apparaître le panneau de configuration, double-cliquer sur un composant ou sur un câble.

Lorsque le panneau de configuration est affiché, un simple clic sur un autre composant permet d'en afficher les paramètres.

Le panneau de configuration se referme lorsque l'on clique sur une zone vide de l'espace de travail.

Pour davantage d'information sur les composants, utiliser le menu en haut de cette page.
