Design mode

Computer and notebook

   

In FILIUS, the Computer and Laptop components have identical functionality. Only their icons are different. Computers and laptops will be designated by the generic term of station.

Each station has an operating system, a desktop and applications which are accessible in simulation mode.

Configuration

Name

The name of a station can be chosen arbitrarily and it doesn't have any influence on the functioning of the simulation.

Display IP address instead of the name

This option allows you to display the station's IP address under its icon instead of the name.

MAC address

This field contains the MAC address of the station. MAC addresses are randomly generated by FILIUS at components creation.

DHCP service

The station can act as a DHCP server. For practical reasons, this service is configured in design mode.

The DHCP service assigns any workstation (other than the one on which it is installed) an IP address and a subnet mask. It can optionally also provide the IP address of a gateway and the IP address of a DNS server.

The current state of the DHCP service is displayed and the Configure button provides access to the parameters of this service.

DHCP service settings

IP address

Field used to enter the station's IP address. This field is required for the station to be able to access the network.

Netmask

Field used to enter the subnet mask of the station. This field is required for the station to be able to access the network.

Gateway

Field allowing the entry of the IP address of the router allowing to leave the local network. This field is only necessary if the station needs to access components located outside its local network.

Domain Name Server

Field used to enter the IP address of a DNS server. This field is required only if certain station applications use domain names (such as a web browser or an email client, for example).

Use DHCP for configuration

Network settings can be obtained automatically from a DHCP server.

In the event of automatic addressing, the station cannot act as a DHCP server.
