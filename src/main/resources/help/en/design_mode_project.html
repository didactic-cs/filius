Design mode

Project

FILIUS saves all the components and their configuration in a project file. These files have the extension .fls.

The saved items are:

The components, their positions, the cables that connect them, their parameters, and, in the case of workstations, the applications installed as well as the files and folders of the workstation's operating system. Structural blocks and texts added in documentation mode are also saved.

The unsaved items are:

The SAT table of switches and the contents of the data tracking window.

Open and save a project

Opening and saving a project can be done from design and documentation modes only. Use the buttons to the left of the top bar for this.

Export as image

The project diagram can be exported as a raster image in PNG format or as a vector image in SVG format. To do this, click on the black arrow on the save button.

Create a report

A project status report can be generated in PDF format. This report consists of the project diagram, the configuration of the components, including the applications installed in the case of workstations and, where applicable, the content of the data tracking window.
