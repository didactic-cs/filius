Design mode

Switch

The Switch component allows multiple components to be interconnected within the same local area network. Its function is to circulate datagrams using the MAC addresses of components on the local network. A switch does not use IP addresses.

Each FILIUS switch has 24 ports which allow the connection of as many components, including other switches.

Settings

Name

The name can be used to differentiate multiple switches, but separate switches can also have the same name. This name has no effect on the simulation.

Display as cloud

This option allows you to change the appearance of the switch on the workspace. It has no effect whatsoever on the simulation.
