General

This mode allows you to add visual elements that make the network easier to read. These elements do not have any influence on the simulation.

In documentation mode, structure fields and text areas are displayed in front of the network components to facilitate their positioning. In other modes, they pass behind the components.

Structure field

Using a drag-and-drop with the mouse, select the structure field in the left margin and drop it on the workspace.

Then move or resize this block using the mouse.

To duplicate a field, click on it to select it, then use the keyboard shortcut Ctrl + D.

To delete a block, click on it, then press the Delete key on the keyboard.

Use the contextual menu of the field (right click) to modify its color.

The stacking order of the fields can be changed using the context menu. This can be useful when two fields overlap.

Text areas

Using a drag-and-drop with the mouse, select the text area in the left margin and drop it on the workspace.

When a text area is selected, it is surrounded by a blue frame. Otherwise, a simple gray outline delimits the area.

To modify the text area, click on it. The text can then be edited directly.

Use the blue frame to move or resize the text area.

To duplicate a text box, click on it to select it, then use the keyboard shortcut Ctrl + D.

To delete the text box, click on it to select it, then use the keyboard shortcut Ctrl + Delete.

Use the context menu to change the font color, size and style of characters.

If necessary, it is possible to change the stacking order of the text areas using the context menu. Note that the text areas are always displayed in front of the structure fields.
