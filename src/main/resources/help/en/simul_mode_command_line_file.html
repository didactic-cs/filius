File commands

The paths to be entered as parameters of certain commands must use the " / " separator.

The character " / " is optional at the end folder paths.

Absolute paths start with " / ".

Paths that do not start with " / " are relative to the current working folder.

touch  file-path

This command creates an empty file.

The file-path parameter must be the name of the file to create, preceded or not by an absolute path or a path relative to the current folder.

copy  source-path  target-path
cp  source-path  target-path

This command makes a copy of a file.

The source-path parameter must be the name of the file to copy, preceded or not by an absolute path or a path relative to the current folder.

The target-path parameter must be the name of the copy to be made, preceded or not by an absolute path or a path relative to the current folder.

move  source-path  target-path
mv  source-path  target-path

This command moves or renames a file.

cat  file-path
type  file-path

This command displays the contents of a file.

mkdir  folder-path

This command creates a folder.

del  path
rm  path

This command deletes a file or an empty folder.

rmdir  folder-path

This command deletes a folder and all of its contents.

cd  folder-path

This command allows you to change the current working folder.

pwd

This command displays the absolute path of the current working folder.

dir  folder-path
ls  folder-path

This command lists the contents of the folder whose path is specified. If there is no parameter, the contents of the current working folder are listed.
