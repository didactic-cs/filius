Network commands

ipconfig

This command displays the configuration of the network interface.

netstat

This command displays information about network connections.

arp

The station's ARP cache is a table containing ARP entries. Each of these entries is a pair made up of an IP address and the corresponding MAC address. The ARP cache is used to speed up the resolution of IP addresses to MAC addresses.

Used without a parameter, or with the -a parameter alone, the arp command displays the contents of the ARP cache. Example of use:
arp -a 192.168.0.10

Used with the -a parameter followed by an IP address, the arp command displays only the entry associated with that address. Example of use:
arp -d 192.168.0.10

Used with the -d parameter followed by an IP address, the arp command removes the entry associated with that address from the cache. Example of use:
arp -d *

Used with the -d parameter followed by the * character, the arp command clears all cache entries.

The cache is also cleared each time the simulation is started.

arpsend  fictitious-ip  destination-ip

This command sends to a destination station a correspondence between the MAC address of the station on which the command is entered and any IP address. This correspondence will be added to the recipient's ARP table.

This command can be used for testing purposes.

host  hostname

This command resolves a hostname to an IP address.

The hostname parameter corresponds to the name of the computer whose IP address is sought.

nslookup  hostname

This command resolves a host name to an IP address. It also specifies the address of the server performing the resolution.

The hostname parameter corresponds to the name of the computer whose IP address is sought.

route

This command displays the routing table of the station.

ping  ip-address | hostname

This command tests the network connection with another station.

The parameter ip-address | hostname can be either the IP address or the name of the other station.

traceroute  ip-address | hostname

This command lists the jumps required to reach a position.

The ip-address | hostname can be either the IP address or the name of the station to be reached.

tcpdump

This command is used to monitor the network traffic by displaying each of the Ethernet frames entering or leaving the station.

It displays the same information as the data tracking window.

To interrupt this command, enter Ctrl+C.
