Mode simulation

Switch

Un commutateur (ou switch) permet d'interconnecter différents élements d'un même sous-réseau.

Un commutateur redirige chaque trame ethernet qui arrive à l'un de ses ports vers le port permettant d'atteindre le destinataire.

Pour éviter de systématiquement diffuser les trames reçues sur tous ses ports, le commutateur mémorise progessivement dans une table les ports correspondant aux adresses MAC des postes. Cette table qui associe aux adresses MAC un numéro de port du commutateur est appelée la table des adresses sources (SAT).

Pour afficher cette table, double-cliquer sur le commutateur.

Les tables SAT des commutateurs sont réinitialisées à chaque démarrage de la simulation.
