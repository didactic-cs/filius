Modes de fonctionnement

       

Filius possède trois modes de fonctionnement.

Mode conception

Dans ce mode, les différents éléments du réseau peuvent être assemblés sur l'espace de travail. Ce mode est celui sélectionné par défaut au démarrage de Filius.

Pour sélectionner ce mode depuis l'un des deux autres, utiliser le raccourci clavier F10 ou cliquer sur le bouton suivant de la barre supérieure :

Mode documentation

Ce mode permet d'ajouter des zones de texte ainsi que des blocs rectangulaires sur l'espace de travail. Ces éléments n'influent pas sur la simulation mais permettent de rendre la lecture du réseau plus facile.

Pour sélectionner ce mode depuis l'un des deux autres, utiliser le raccourci clavier F9 ou cliquer sur le bouton suivant de la barre supérieure :

Mode simulation

Ce mode permet de simuler le fonctionnement du réseau. La simulation comprend les échanges de données entre les différents éléments qui constituent réseau, ainsi que les interactions avec les applications des postes virtuels de ce réseau.

Pour sélectionner ce mode depuis l'un des deux autres, utiliser le raccourci clavier F11 ou cliquer sur le bouton suivant de la barre supérieure :
