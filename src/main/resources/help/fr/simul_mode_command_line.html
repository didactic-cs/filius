Mode simulation

Ligne de commandes

Cette application permet la saisie de commandes permettant de manipuler les fichiers et d'effectuer des opérations liées au réseau. Chaque commande doit être saisie suivie de ses éventuels paramètres et terminée par la touchée entrée.

Certaines commandes possèdent deux syntaxes.

help

Cette commande sans paramètre affiche la liste des commandes disponibles.

touch  chemin

Cette commande crée un fichier vide.

Le paramètre chemin doit être le nom du fichier à créer, précédé ou non d'un chemin absolu ou relatif au dossier courant.

copy  source  destination
cp  source  destination

Cette commande effectue une copie d'un fichier.

Le paramètre source doit être le nom du fichier à copier, précédé ou non d'un chemin absolu ou relatif au dossier courant.

Le paramètre destination doit être le nom du fichier à copier, précédé ou non d'un chemin absolu ou relatif au dossier courant.

move  source  destination
mv  source  destination

Cette commande déplace ou renomme un fichier.

cat  chemin
type  chemin

Cette commande affiche le contenu d'un fichier.

del  chemin
rm  chemin

Cette commande supprime un fichier ou un dossier.

mkdir  chemin

Cette commande crée un dossier.

cd  chemin

Cette commande permet de changer le dossier de travail courant.

pwd  chemin

Cette commande permet d'afficher le chemin absolu du dossier de travail courant.

dir  chemin
ls  chemin

Cette commande liste le contenu du dossier de travail courant.

ipconfig

Cette commande affiche la configuration de l'interface réseau.

netstat

Cette commande affiche des informations sur les connexions réseau.

arp

Cette commande affiche la table de correspondance entre les adresses MAC et les adresses IP.

host  nom_d_hote

Cette commande résout un nom d'hôte en adresse IP.

Le paramètre nom_d_hote correspond au nom du poste dont l'adresse IP est recherchée.

route

Cette commande affiche la table de routage du poste.

ping  adresse_ip/nom_d_hote

Cette commande teste la connexion réseau avec un autre poste.

Le paramètre adresse_ip/nom_d_hote correspond, au choix, à l'adresse IP ou au nom de l'autre poste.

traceroute  adresse_ip/nom_d_hote

Cette commande liste les sauts nécessaires pour atteindre un poste.

Le paramètre adresse_ip/nom_d_hote correspond, au choix, à l'adresse IP ou au nom du poste à atteindre.

exit

Cette commande ferme la fenêtre de l'application.
