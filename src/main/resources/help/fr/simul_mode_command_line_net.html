Commandes relatives au réseau

ipconfig

Cette commande affiche la configuration de l'interface réseau.

netstat

Cette commande affiche des informations sur les connexions réseau.

arp

Le cache ARP du poste est une table contenant des entrées ARP. Chacune de ces entrées est un couple constitué d'une adresse IP et de l'adresse MAC correspondante. Le cache ARP est utilisé pour accélérer la résolution d'adresses IP en adresses MAC.

Utilisée sans paramètre, ou avec le paramètre -a seul, la commande arp affiche le contenu du cache ARP.

Utilisée avec le paramètre -a suivi d'une adresse IP, la commande arp affiche uniquement l'entrée associée à cette adresse.

Utilisée avec le paramètre -d suivi d'une adresse IP, la commande arp supprime du cache l'entrée associée à cette adresse.

Utilisée avec le paramètre -d suivi du caractère *, la commande arp efface toutes les entrées du cache.

Le cache est par ailleurs réinitialisé à chaque démarrage de la simulation.

arpsend  ip-fictive  ip-destinataire

Cette commande envoie à un poste destinataire l'entrée ARP constituée d'une adresse IP arbitraire et de l'adresse MAC du poste sur lequel la commande est saisie. Cette entrée sera ajoutée à la table ARP du destinataire.

Cette commande peut être utilisée à des fins de tests.

host  nom-d-hote

Cette commande résout un nom d'hôte en adresse IP.

Le paramètre nom-d-hote correspond au nom du poste dont l'adresse IP est recherchée.

nslookup  nom-d-hote

Cette commande résout un nom d'hôte en adresse IP. Elle précise également l'adresse du serveur qui effectue la résolution.

Le paramètre nom-d-hote correspond au nom du poste dont l'adresse IP est recherchée.

route

Cette commande affiche la table de routage du poste.

ping  adresse-ip | nom-d-hote

Cette commande teste la connexion réseau avec un autre poste.

Le paramètre adresse-ip | nom-d-hote correspond, au choix, à l'adresse IP ou au nom de l'autre poste.

traceroute  adresse-ip | nom-d-hote

Cette commande liste les sauts nécessaires pour atteindre un poste.

Le paramètre adresse-ip | nom-d-hote correspond, au choix, à l'adresse IP ou au nom du poste à atteindre.

tcpdump

Cette commande permet de suivre le trafic réseau en affichant chacune des trames Ethernet entrant ou sortant du poste.

Elle affiche les mêmes informations que la fenêtre de suivi des données.

Pour interrompre cette commande, saisir Ctrl+C.
