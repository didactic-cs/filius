Mode simulation

Switch

Un commutateur (ou switch) enregistre les adresses MAC des paquets entrants avec le port d’arrivée. Ces deux informations sont stockées dans la table des adresses sources (SAT). Pour afficher cette table, double-cliquer sur le commutateur.
