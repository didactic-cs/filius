Mode conception

Projet

FILIUS sauvegarde l'ensemble des composants et leur configuration dans un fichier projet. Ces fichiers ont pour extension .fls.

Les éléments sauvegardés sont :

Les composants, leurs positions, les câbles qui les relient, leurs paramètres, et, dans le cas des postes, les applications installées ainsi que les fichiers et dossiers du système d'exploitation du poste. Les blocs structurels et les textes ajoutés en mode documentation sont également sauvegardés.

Les éléments non sauvegardés sont :

La table SAT des commutateurs et le contenu de la fenêtre de suivi des données.

Ouvrir et enregistrer un projet

L'ouverture et la sauvegarde d'un projet peuvent être effectués à partir des modes conception et documentation uniquement. Utiliser pour cela les boutons situés à gauche de la barre supérieure.

Exporter en tant qu'image

Le schéma du projet peut être exporté sous forme d'image matricielle au format PNG ou sous forme d'image vectorielle au format SVG. Pour cela, cliquer sur la flèche noire du bouton d'enregistrement.

Générer un rapport

Un rapport de l'état du projet peut être généré au format PDF. Ce rapport est constitué du schéma du projet, de la configuration des composants, y compris les applications installées dans le cas des postes et, le cas échéant, du contenu de la fenêtre de suivi.
