Mode documentation

Ce mode permet d’ajouter des éléments visuels qui facilitent la lecture du schéma du réseau. Ces éléments n'ont aucune influence sur la simulation.

Dans le mode documentation, les blocs structurels et les zones de texte sont affichés devant les composants du réseau. Dans les autres modes, ils repassent derrière les composants.

1. Blocs structurels

À l’aide d’un glisser-lâcher à la souris, sélectionner le bloc structurel dans la marge de gauche et le déposer sur l’espace de travail.

Déplacer ou redimensionner ensuite ce bloc à l'aide de la souris.

Pour dupliquer un bloc, cliquer dessus pour le sélectionner, puis utiliser le raccourci clavier Ctrl + D.

Pour supprimer un bloc, cliquer dessus, puis appuyer sur la touche Suppr du clavier.

Utiliser le menu contextuel du bloc (clic droit) pour modifier sa couleur.

L'ordre d'empilement des blocs peut être modifié en utilisant le menu contextuel. Cela peut s'avérer utile lorsque deux blocs se chevauchent.

2. Zones de texte

À l’aide d’un glisser-lâcher à la souris, sélectionner la zone de texte dans la marge de gauche et la déposer sur l’espace de travail.

Lorsqu'une zone de texte est sélectionnée, elle est entourée par un cadre bleu. Dans le cas contraire, un simple contour gris délimite la zone.

Pour modifier la zone de texte, cliquer dessus. Le texte peut alors être édité directement.

Utiliser le cadre bleu pour déplacer ou redimensionner la zone de texte.

Pour dupliquer une zone de texte, cliquer dessus pour la sélectionner, puis utiliser le raccourci clavier Ctrl + D.

Pour supprimer la zone de texte, cliquer dessus pour la sélectionner, puis utiliser le raccourci clavier Ctrl + Suppr.

Utiliser le menu contextuel pour changer la couleur, la taille et le style de la police de caractères.

Si nécessaire, il est possible de modifier l'ordre d'empilement des zones de texte à l'aide du menu contextuel. Il est à noter que les zones de texte sont toujours affichées devant les blocs structurels.
